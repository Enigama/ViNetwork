# Privacy Policy for ViNetwork

**Last updated:** December 26, 2024

## Overview

ViNetwork is a Chrome DevTools extension that provides Vim-style keyboard navigation for the Network tab. This privacy policy explains how we handle your data.

## Data Collection

**We do not collect any data.**

ViNetwork:
- Does NOT collect personal information
- Does NOT track your browsing activity
- Does NOT send any data to external servers
- Does NOT use analytics or telemetry
- Does NOT store any data outside your browser

## Data Storage

ViNetwork uses Chrome's local storage API (`chrome.storage.local`) solely to save your preferences (such as settings). This data:
- Stays on your device
- Is never transmitted anywhere
- Can be cleared by removing the extension

## Network Requests

ViNetwork captures network request data only within Chrome DevTools for display purposes. This data:
- Is only visible to you in DevTools
- Is never sent to any external service
- Is cleared when you close DevTools or clear requests

## Permissions

ViNetwork requests the following permissions:
- **debugger**: Required to capture network requests in DevTools
- **storage**: Required to save your local preferences
